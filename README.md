# Blog Image Repository

This repository stores images for my Hexo blog (https://raidenxe.github.io/).

## Usage

Images are served via GitHub Pages or jsDelivr CDN.

### CDN URL Format

```
https://cdn.jsdelivr.net/gh/Raidenxe/blog-img@main/path/to/image.jpg
```

### GitHub Pages URL Format

```
https://raidenxe.github.io/blog-img/path/to/image.jpg
```

## Folder Structure

```
├── post/          # 文章图片
│   └── 2026/      # 按年份分类
├── cover/         # 封面图片
└── banner/        # 横幅图片
```
